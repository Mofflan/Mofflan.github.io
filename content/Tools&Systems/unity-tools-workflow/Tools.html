Easy start of the day tool button

I created a "Daily Click" export button to solve a painful repetition: we used Unity purely as a level editor and exported level data (JSON) into our Friendship Engine. Unity regenerated instance IDs each day, so manually exporting every level one by one was slow and error-prone. The Daily Click button iterated all level scenes and produced fresh JSON for each in one go— including scene name, entity instance IDs, object names, transforms, box collider data, and other metadata. This let the whole team press one button at the start of the day and immediately sync consistent data, instead of doing dozens of individual exports.
