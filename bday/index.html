Intro

In this project I worked on the Player movement (its just a camera that moves and rotate). Additionally, I developed various tools within Unity to streamline the workflow for other team members, making it easier for them to contribute effectively to the project.

 

Showcasing Tools

 

This tool simplifies the game startup process by initiating gameplay with the click of a button. It offers the flexibility to select between debug, release, or retail modes, automatically building the project before launching the executable file.
